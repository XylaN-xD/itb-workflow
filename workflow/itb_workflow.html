Block 05
Few-Shot Annotation + YOLOv8 Fine-Tuning
Manual gastroenterologist labels · Frozen CLIP backbone · Crohn's as hard negatives
Annotation Strategy (No Synthetic)
Image-level binary labels — cheapest, 1/20th cost of dense seg
Coarse bounding boxes — 30–90 sec / image
Real samples only: Kvasir-Capsule · HyperKvasir
YOLOv8 Architecture
Frozen CLIP backbone + YOLOv8 detection head
Orientation-aware loss: transverse vs longitudinal ulcers
Crohn's disease cases as hard negatives
⚠ KEY DIFFERENTIAL: ITB shows transverse ulcers at ileocaecal junction. Crohn's shows longitudinal ulcers. Model must learn orientation-sensitive features.
◆ Decision Gate — Fine-Tuned F1 ≥ 0.91?
YES → explainability layer  ·  NO → collect more annotations (20–50 per round, 4–6 week cadence) + retrain
